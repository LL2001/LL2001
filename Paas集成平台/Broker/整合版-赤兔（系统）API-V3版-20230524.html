创建非分布式实例

  • 请求参数：
名称 类型 必须 描述
请求参数 object 是
Action string 是 固定赋值：CreateNoShard
ApiModule string 是 固定赋值：c_tdsql
ClusterKey string 是 集群ID
BusinessDesc string 是 业务描述
InstanceName string 是 实例名
DbVersion string 是 安装的数据库版本信息，在OSS配置文件中配置，默认值在manage中配置.
Arch string 是 CPU架构
Machine string 是 部署机型
CPU number 是 CPU规格（100=1核）
Memory number 是 内存规格（1000MB=1GB）
DataDisk number 是 数据盘规格（1000MB=1GB）
LogDisk number 是 日志盘规格（1000MB=1GB）
NodeNum number 是 实例节点数
1：一主零备
2：一主一备
3：一主两备（默认）
4：一主三备
5：一主四备
6：一主五备
SyncType number 是 主机和备机的模式
0：异步；
1：强同步；
2：同IDC异步，不同IDC强同步，
3：跨AZ强同步，同AZ异步模式
Degrade number 是 退化标识；
0：不退化（默认）
1：退化；
例如：当购买的set成为单主(两个备机同时挂掉)，当设置为1的时候，会退化成异步模式，仍可提供服务。否则强同步的时候，会得不到备机的应答，写操作会失败。
FenceId string 是 独享集群标识，none为公有云，其他为独享集群标识，如果创建集群的时候，指定了fenceid，则添加实例的时候，必须与之相同。默认是none。 需要接口获取api/tdsql2/fence/searchList
ProxyGroupId string 是 网关组ID
DomainType string 是 接入配置类型，可以是lvs , dns , l5 , tgw其中的一个。如果有多个中间用分号(;)隔开
Domain string 是 接入配置值
Ipv6Flag number 是 是否开启IPV6（1：assign_ip应当为ipv6列表，0：assign_ip应当为ipv4列表）
IdcFlag boolean 是 是否检查idc信息，作为标志,默认是true。如果检查的话，要求assign_ip的各个ip分别在不同的idc才行
AssignIp any[] 是 服务器分配。 如果指定了资源信息，则使用该字段标识当前那些机器ip；例如：192.168.1.2;192.168.1.3;192.168.1.4
MasterZone string 是 主db所在区。 分离版本不需要传该参数， 合并版本才有这个参数 api/tdsql2/device/zoneList
SlaveZones any[] 是 备机的DB所在的区，多个区用分号隔开. api/tdsql2/device/zoneList获取列表
ResAssignMode object[] 是 支持相同IDC机型组建set；res_assign_mode:"none:1;none:2"表示1台机器在某个idc，另外2台在另外一个idc。 api/tdsql2/idc/searchList获取列表
IDC string 是 IDC名称
DBNum number 是 IDC部署节点数量
CharacterSetServer string 是 字符编码 utf8mb4、utf8
CollationServer string 是 排序规则 utf8mb4_general_ci 、utf8_general_ci
LowerCaseTableNames string 是 表名大小写 “0” 敏感 “1”不敏感
InnodbPageSize string 是 innodb_page_sizea 默认：16384
  • 示例：
{"Action":"CreateNoShardInfo","ApiModule":"c_tdsql","ClusterKey":"tdsql2_sep1","BusinessDesc":"业务描述","InstanceName":"实例名称","DbVersion":"8.0.24","Arch":"X86","Machine":"TS85","CPU":100,"Memory":1000,"DataDisk":24000,"LogDisk":24000,"NodeNum":2,"SyncType":1,"Degrade":1,"FenceId":"none","ProxyGroupId":"GROUP_0000000000","DomainType":"","Domain":"","Ipv6Flag":0,"IdcFlag":true,"AssignIp":[],"MasterZone":"","SlaveZones":[],"ResAssignMode":[{"IDC":"IDC1","DBNum":1},{"IDC":"IDC2","DBNum":1}],"CharacterSetServer":"utf8mb4","CollationServer":"utf8mb4_general_ci","LowerCaseTableNames":"1","InnodbPageSize":"16384"}

名称 类型 必须 描述
